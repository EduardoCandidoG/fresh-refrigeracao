# Fresh – Sistema de Gestão de Serviços de Refrigeração

Projeto para a disciplina de Processo de Desenvolvimento de Software,
com foco na aplicação da metodologia ágil Scrum.

## Objetivo
Auxiliar microempreendedores do ramo de refrigeração no controle de clientes,
orçamentos, agendamentos e serviços realizados.

## Linguagens e ferramentas que serão utilizadas
  Frontend: HTML, CSS, JavaScript
  Backend: Java, Spring Boot
  Banco de dados: Postgre
  Ferramentas: Git e Github
